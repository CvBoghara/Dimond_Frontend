# Diamond Management Frontend

## Tech Stack

* React.js
* Axios
* React Router DOM

## Features

* Dashboard
* Work Entry Management
* Salary Report
* Employee List
* Pagination
* Authentication (Demo Login)

## Setup

```bash
npm install
npm run dev
```
